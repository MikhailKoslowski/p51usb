P51USB

Kit didático para ensino de microcontroladores

Sobre a P51USB

A P51USB é um kit didático de baixo custo que foi desenvolvido para substituir o kit P51N utilizado nas disciplinas de microcontroladores na UTFPR Curitiba.

As principais melhorias com relação ao kit anterior são:

  • Memória flash interna de 32k para programa, além de 1k de EEPROM para dados;
  • Periféricos SPI e I2C na placa, com controlador integrado ao 8051;
  • LEDs e chaves montadas na placa para experimentos simples de IO;
  • Interface USB nativa;
  • Bootloader via USB para gravação do microcontrolador, dispensando o uso de gravadores externos;
  • Os ports 0 e 2 não estão alocados para acesso a memória externa e portanto podem ser livremente utilizados.

Além disso, algumas caracteristicas originais do kit anterior foram preservadas:

  • Barramento pronto para conexão de display LCD 16x2;
  • Pinos dos ports disponíveis em barras simples para fácil acesso e utilização;
  • Possibilidade de depuração utilizando FlashMon51 (versão Flash do Monitor51) via porta Serial.
